## 🎯 Что такое TRADEAR?

TRADEAR — AI-консультант по торговле и 24/7 автоматизированная торговля на криптобиржах.

**Три режима:**
- 📱 **Консультант** (Бесплатно) — анализ чартов TradingView в Claude.ai
- 💾 **Paper Trading** (Бесплатно) — тестирование стратегий на исторических данных
- 🖥️ **Автотрейдер** (Платно $29–49/мес) — 24/7 автоматизированная торговля на VPS

---

## ✨ Возможности

- Технический анализ (паттерны, поддержка/сопротивление, тренды)
- Управление рисками (стоп-лосс, тейк-профит, размер позиции)
- 4 встроенные стратегии (Тренд-фолоуинг, DCA, Скальпинг, Сетка)
- 5 бирж (Binance, Coinbase, Kraken, Bybit, OKX)
- Мониторинг в реальном времени

---

## 🚀 Быстрый старт

1. Скопируй промпт из `tradear-base-prompt.txt`
2. Вставь в [Claude.ai](https://claude.ai)
3. Покажи чарт TradingView или опиши пару
4. Получи сигнал BUY/SELL/HOLD с входом, целью, стоп-лоссом
